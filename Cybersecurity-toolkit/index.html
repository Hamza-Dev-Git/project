Field Kit for
Security Practitioners.

A working reference of the tools most commonly reached for across the pentest and blue-team lifecycle — from first scan to final report. Use only against systems you own or are explicitly authorized to test.

Recon & Analysis Offensive Testing Platform
Recon

Nmap

Network Mapper

The standard for host discovery and port scanning — sweeps networks to find live hosts, open ports, running services, and OS fingerprints before any deeper testing begins.

Capabilities

  • Host & port discovery across large IP ranges
  • Service and OS version detection
  • Scriptable engine (NSE) for custom checks
  • Stealth, TCP, UDP, and timing-tuned scan modes
$ nmap -sV -A target.local Official Website ↗
Analysis

Wireshark

Packet Analyzer

A deep-inspection packet capture tool that decodes traffic across hundreds of protocols, letting analysts see exactly what's moving across a network in real time or from a saved capture.

Capabilities

  • Live capture and offline .pcap analysis
  • Protocol dissection for 1000+ protocols
  • Custom display and capture filters
  • Stream reassembly for full conversations
$ wireshark -i eth0 -k Official Website ↗
Web

Burp Suite

Web Application Testing

An intercepting proxy at the center of most web app assessments — inspects and modifies traffic between browser and server, and automates crawling for common vulnerabilities.

Capabilities

  • Intercepting proxy for live request tampering
  • Automated crawler and vulnerability scanner
  • Repeater & Intruder for manual and fuzzed testing
  • Extensible via the BApp marketplace
$ burpsuite --project-file=scan.burp Official Website ↗
Exploit

Metasploit

Exploitation Framework

A modular framework for developing, testing, and running exploits against known vulnerabilities — used to validate whether a weakness is actually exploitable in a controlled environment.

Capabilities

  • Large, regularly updated exploit & payload library
  • Post-exploitation and session management
  • Meterpreter for interactive remote control
  • Integrates with Nmap and Nessus scan data
$ msfconsole -q Official Website ↗
Scan

Nessus

Vulnerability Scanner

A widely deployed vulnerability scanner that checks hosts against a huge, continuously updated plugin library to surface missing patches, misconfigurations, and known CVEs.

Capabilities

  • 200,000+ plugin checks across systems & software
  • Compliance and configuration auditing
  • Scheduled scans with prioritized risk scoring
  • Exportable, report-ready findings
$ nessuscli scan --new Official Website ↗
Platform

Kali Linux

Penetration Testing OS

A Debian-based Linux distribution built specifically for security work, shipping with hundreds of testing tools pre-installed and configured so assessments can start immediately.

Capabilities

  • 600+ pre-installed security tools
  • Live-boot, persistent USB, and ARM builds
  • Rolling release with frequent tool updates
  • Purpose-built for pentesting & forensics
$ uname -a Official Website ↗
Web

OWASP ZAP

Zed Attack Proxy

A free, open-source web app scanner maintained by OWASP — a lighter, community-driven alternative to commercial proxies, built for both beginners and CI-integrated automated scanning.

Capabilities

  • Automated and manual web app scanning
  • Passive scanning safe for production traffic checks
  • REST API for CI/CD pipeline integration
  • Free and fully open-source
$ zap.sh -cmd -quickurl target Official Website ↗
Crack

John the Ripper

Password Cracker

A fast, offline password-cracking tool used to test password strength by recovering weak hashes — a staple for auditing credential policies rather than attacking live logins.

Capabilities

  • Auto-detects hash types across formats
  • Dictionary, brute-force, and rule-based attacks
  • GPU-accelerated cracking support
  • Wide platform and hash-format support
$ john --wordlist=rockyou.txt hash.txt Official Website ↗
Crack

Hydra

Login Brute-Forcer

A parallelized online login-cracking tool that tests credential lists against live services — SSH, FTP, HTTP forms, and dozens more — to check for weak or default passwords.

Capabilities

  • Supports 50+ protocols and services
  • Highly parallel, multi-threaded attacks
  • Works with custom user/password wordlists
  • Scriptable for repeatable credential audits
$ hydra -l admin -P list.txt ssh://target Official Website ↗
Authorized use only. These tools are built for defenders, auditors, and testers working within the scope of written permission. Running them against systems you don't own or aren't authorized to assess is illegal in most jurisdictions.
